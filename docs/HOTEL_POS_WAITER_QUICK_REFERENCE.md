# Hotel POS Waiter Quick Reference

Use this sheet during service for fast, consistent POS handling.

## 1) Open Table Session

- Go to `Sales -> Hotel POS`.
- Select table (`T1`, `VIP1`, etc.).
- Click `Open` in session controls.

## 2) Take and Build Order

- Choose `Department` and `Menu`.
- Search and click items to add to cart.
- Update quantity using `+` / `-`.
- Add comments (example: `no salt`, `extra hot`).
- Set course/timing when needed.

## 3) Before Posting (Always Check)

- Correct table number.
- Correct items and quantities.
- Correct total amount.
- Customer selected (if required).

## 4) Action Buttons (When to Use)

- `Send to Kitchen`: send for preparation only.
- `Pay Now`: guest pays now (choose payment method first).
- `Bill to Room`: charge to selected room guest.
- `Credit Sale`: record now, payment later.
- `Hold Ticket`: pause order and resume later.
- `Print Bill`: print bill for guest.

## 5) Queue and Edits

- Check `Order Queue` for active orders.
- Use filters (`Status`, `Station`) to find orders quickly.
- Use `Edit` only for valid corrections.
- Save edits immediately.

## 6) Important Rules

- Do not post same payment twice.
- Do not share manager PIN.
- For refund/void/locked actions, call supervisor/manager.
- Do not close session if orders are still unsettled.

## 7) End of Shift Mini Checklist

- No open unsent cart.
- All served orders posted correctly.
- Payments match cash/card/mobile totals.
- Completed sessions closed.
