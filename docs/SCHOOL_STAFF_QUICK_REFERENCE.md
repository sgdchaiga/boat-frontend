# School Staff Quick Reference

Use this as a daily quick sheet for front-office, bursar support, and academic staff.

## 1) Student Records

- Search first before creating a new student.
- Capture full guardian contact details.
- Confirm class/stream assignment is correct.

## 2) Fees and Payments

- Confirm student and fee item before posting payment.
- Select correct payment method and reference.
- Issue receipt immediately.
- For discount/waiver/refund, escalate for approval.

## 3) Attendance

- Mark attendance on time for assigned class.
- Correct errors the same day.
- Report unresolved class attendance gaps to admin.

## 4) Academic Entries

- Enter marks only for your assigned class/subject.
- Validate score ranges before save.
- Recheck report outputs before release.

## 5) Daily Must-Do

- No duplicate student or payment entries.
- No unreceipted payments.
- No shared credentials.
- Raise issues quickly to supervisor/manager.

## 6) End-of-Shift Mini Checklist

- Transactions posted and confirmed.
- Attendance completed (if assigned).
- Pending tasks handed over with notes.
