## Recommended Rollout Order

1. Share `Hotel POS SOP` with all hotel operations staff.
2. Train waiters/front service using the waiter quick reference.
3. Train supervisors/managers using the manager quick reference.
4. Run a short practical session:
   - Open session
   - Create order
   - Send to kitchen
   - Pay now / bill to room
   - Print bill
   - Review posted transactions

## Suggested Training Checklist

- Staff can create and post a complete order correctly.
- Staff can explain each action button in Hotel POS.
- Waiters know when manager approval is required.
- Managers can review edits/voids/refunds with reasons.
- End-of-shift reconciliation process is understood.
