# School Manager Quick Reference

Use this guide for school-level oversight, financial control, and approvals.

## 1) Manager Responsibilities

- Ensure data quality in student, class, and finance records.
- Enforce role-based access and accountability.
- Approve only valid exceptions with clear reasons.
- Monitor daily operational and financial health.

## 2) Approval Controls

- Approve sensitive actions only after verification:
  - Fee adjustment
  - Waiver/discount override
  - Refund/reversal
  - Back-dated transaction edits
- Require reason notes on all approvals.
- Reject incomplete requests (missing learner or payment evidence).

## 3) Financial Oversight

- Review daily collections by payment method.
- Confirm receipts align with posted payments.
- Validate outstanding balances and aging trends.
- Investigate duplicate or suspicious transactions immediately.

## 4) Academic and Attendance Oversight

- Monitor attendance completion rates by class.
- Ensure unresolved attendance anomalies are closed quickly.
- Validate marks/report release workflow and approvals.

## 5) Security and Compliance

- No credential sharing.
- Ensure each action is attributable to a real user account.
- Lock down unauthorized role privileges.
- Escalate suspected data leakage or unauthorized edits immediately.

## 6) End-of-Day Manager Checklist

- Collections reconciled to system totals.
- Pending approvals resolved or documented.
- Attendance completion reviewed.
- Exception log updated for follow-up.
- Critical incidents escalated to owner/finance/admin.

## 7) Escalation Triggers

- Missing receipts for posted payments.
- Unexpected balance changes on student accounts.
- Repeated refund or waiver requests by same user.
- Back-dated edits without approval trail.
- Cross-organization data visibility anomaly.
